CCI Toolbox

Welcome to Cate, the Climate Analysis Toolbox for the ESA Climate Change Initiative.

.

Cate is a software environment for ingesting, operating on and visualising data from all ECV projects of the Climate Change Initiative (CCI).

@

!!!

Cate works by mashing all the different types of ECV data into a Common Data Model.

@

!!!

Users then apply operations, including their own, to that Common Data Model.

@

!!!

Visualise them too.

@

!!!

Cate is Open Source and was first made available on Github in January 2016. We will be releasing Cate Version 1.0 soon.

@

!!!

There are three ways to use Cate.

A Command Line Interface

Our templates are updated regularly so they don't break.

A Graphical User Interface

You can use this theme as is, or you can make changes!

A Cate Python API

We update dependencies to keep things fresh.

We will be releasing training material to help you along the way. Including YouTube videos. Here is everything you need to know to Getting Started
