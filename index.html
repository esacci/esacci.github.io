Welcome to Cate, the Climate Analysis Toolbox for the ESA Climate Change Initiative.
Cate is a software environment for ingesting, operating on and visualising data from all ECV projects of the Climate Change Initiative. It does this by mashing the different types of ECV data into a Climate Data Model. Users then apply operations, including their own, to that data. Visualise them too.
Cate is Open Source and was first made available on Github in January 2016. We will be releasing Cate Version 1.0 ("Offenbach") soon.
There are 3 ways to use Cate. A Graphical User Interface (GUI). A Command Line Interface (CLI). A Cate Python API. Find out more through our documentation.
We will be releasing training material to help you along the way. Including YouTube videos. Here is everything you need to know to Getting Started.
