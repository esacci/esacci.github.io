CCI Toolbox
Welcome to Cate, the Climate Analysis Toolbox for the ESA Climate Change Initiative.

About

Cate is a software environment for ingesting, operating on and visualising all ESA Climate Change Initiative data.

Cate works by mashing ECV data into a Common Data Model.
Users operate on this model,
then visualise the results.

Cate Interfaces

A Graphical User Interface

A desktop application providing a graphical interface to the same functionality provided by the Command Line Interface.

A Command Line Interface

Access and process ESA climate data through a command shell or console terminal, and can also be used to write batch scripts.

A Python API.

The Cate Python API allows you to use Cate in your own Python programs, and make up new functions for Cate too.

Cate is Open Source.
Version 1.0 is now available.
Includes installers for
Windows, Linux & Mac.


Whether you're on Mac, Windows or Linux, how you use Cate is pretty much the same. First follow the download instructions below, then let's talk about documentation.

  • Download Cate On Github
    Go to the Cate releases page on Github. The latest Cate release will be at the top. Download both Cate Core and Cate Desktop.
  • Install Cate Core On your computer
    Install Cate Core first, before Cate Desktop. If you're on Mac or Linux, do this:
    bash cate-'Version'-'Platform'.sh
  • Install Cate Desktop On your computer
    Execute the installer. No bash scripts this time if you're on Mac or Linux.

Now use our documentation.
From how to get started.
